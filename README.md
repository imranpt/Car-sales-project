# Car Sales System

A simple C program for managing car inventory and sales.

## Features

- View available cars in stock
- Buy cars (10% discount for orders > 2 cars)
- View sales history
- Collect customer feedback
- Auto-save sales data

## Build & Run

# Windows
cmd
gcc main.c -o main.exe
main.exe

# linux/macOS
gcc main.c -o main
./main
